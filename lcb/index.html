Little Cree Books for the World

This project aims to make available the source texts and translations of stories from Little Cree Books in a variety of formats. The orginal stories are written in Plains Cree, Swampy Cree, and English, and can be found on the LCB Source homepage.

Links

  • LCB Source - All the source texts in markdown format
  • LCB Imagebank - Repository of open-licensed images from Little Cree Books
  • Global LCB - Repository of translations of LCB stories into other languages
  • Little Cree Books website

License

All stories and images have been released under a CC BY-NC-SA license, allowing them to be shared and modified freely for non-commercial purposes.
